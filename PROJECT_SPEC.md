# 🌌 Pixel Writing Universe - 沉浸式像素风写作宇宙

## 📋 项目概述

一个通过写作养成植物与数字房间的沉浸式创作平台。用户写字越多，植物会成长、动物会互动、房间会变化。

### 核心体验
- **温暖** - 治愈系视觉风格
- **陪伴** - 数字宠物陪伴写作
- **成长** - 游戏化成长机制
- **沉浸** - 动态房间氛围

---

## 🎮 游戏化机制设计

### 1. 经验值系统

| 行为 | XP奖励 |
|------|--------|
| 每写100字 | +5 XP |
| 连续写作30分钟 | +20 XP |
| 每日首次写作 | +10 XP |
| 完成一篇文章 | +50 XP |
| 连续写作7天 | +100 XP |

### 2. 等级系统

```
等级 1-5  : 种子阶段
等级 6-10 : 成长阶段  
等级 11-15: 成熟阶段
等级 16-20: 魔法阶段
等级 21+  : 传说阶段
```

### 3. 成就系统

| 成就 | 描述 | 奖励 |
|------|------|------|
| 🌱 初露锋芒 | 累计写作1000字 | 解锁多肉植物 |
| 🌿 茁壮成长 | 累计写作5000字 | 解锁柴犬宠物 |
| 🌸 花开富贵 | 累计写作10000字 | 解锁樱花树 |
| 📚 著作等身 | 累计写作50000字 | 解锁书房主题 |
| 🔥 连续七日 | 连续写作7天 | 解锁火焰特效 |
| ✨ 夜猫子 | 凌晨2点后写作 | 解锁夜景主题 |

### 4. Streak 系统

- 连续写作天数记录
- 每日重置时间：凌晨4点
- Streak奖励：每日+5% XP加成

## 🌿 植物成长系统

### 植物类型

| 植物 | 解锁条件 | 成长周期 | 特殊效果 |
|------|----------|----------|----------|
| 🌵 多肉 | 初始解锁 | 5000字 | 耐旱，无需浇水 |
| 🌸 樱花 | 等级5 | 15000字 | 开花时房间飘落花瓣 |
| 🍄 蘑菇 | 等级8 | 8000字 | 夜间发光 |
| ✨ 魔法植物 | 等级12 | 30000字 | 随机产生增益 |
| 💡 发光植物 | 等级15 | 40000字 | 照亮房间 |
| 🪴 藤蔓 | 等级18 | 50000字 | 攀爬墙壁装饰 |
| 🎋 竹子 | 等级20 | 60000字 | 竹林音效 |

### 成长阶段

```
阶段1: 种子 → 发芽 (0-1000字)
阶段2: 幼苗 → 长叶 (1000-5000字)  
阶段3: 成长 → 开花 (5000-15000字)
阶段4: 成熟 → 发光 (15000-30000字)
阶段5: 稀有 → 传说形态 (30000+字)
```

### 成长动画

1. **发芽** - 地面裂开，小芽冒出
2. **长叶** - 叶子展开动画
3. **开花** - 花朵绽放粒子效果
4. **发光** - 柔和光晕扩散
5. **稀有** - 彩虹环绕特效

## 🐈 动物陪伴系统

### 动物类型

| 动物 | 解锁条件 | 个性 | 互动动作 |
|------|----------|------|----------|
| 🐱 黑猫 | 初始解锁 | 高冷 | 睡觉、舔毛、跳跃 |
| 🐕 柴犬 | 等级3 | 热情 | 摇尾巴、转圈、扑球 |
| 🐧 企鹅 | 等级6 | 呆萌 | 滑行、吃鱼、摇摆 |
| 🦊 白狐 | 等级10 | 神秘 | 发光、瞬移、望月 |
| 🪼 水母 | 等级14 | 梦幻 | 漂浮、发光、变形 |
| 👻 小幽灵 | 等级18 | 调皮 | 穿墙、闪烁、恶作剧 |

### 互动状态机

```
待机 → 观察 → 靠近 → 互动 → 离开 → 待机
     ↓                    ↑
   睡觉 ←━━━━━━━━━━━━━━━━━━
```

### 写作触发互动

- 每写500字：动物做一个互动动作
- 每写2000字：动物展示特殊技能
- 连续写作30分钟：动物跳上书桌陪伴

## ✨ 动画方案

### CSS动画

1. **植物成长动画**
   - `grow` - 缩放+透明度变化
   - `bloom` - 花朵绽放
   - `glow` - 发光脉动

2. **宠物动画**
   - `idle` - 呼吸起伏
   - `walk` - 左右移动
   - `jump` - 跳跃效果
   - `sleep` - 睡眠动画

3. **房间动画**
   - `rain` - 雨滴下落
   - `snow` - 雪花飘落
   - `twinkle` - 星星闪烁
   - `float` - 漂浮效果

### 粒子系统

```typescript
interface Particle {
  id: string;
  x: number;
  y: number;
  size: number;
  speed: number;
  opacity: number;
  color: string;
  type: 'petal' | 'rain' | 'snow' | 'sparkle';
}
```
