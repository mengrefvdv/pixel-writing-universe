# ✍️ Pixel Writing Universe

> 为享受孤独的人打造的沉浸式像素风写作平台

一个融合游戏化元素的写作工具，通过养成植物、宠物和打造专属房间，让写作变得更有趣、更有成就感。

---

## 🌟 项目特色

### 🎮 游戏化写作体验
- **植物成长系统** - 写作越多，植物成长越快
- **宠物陪伴系统** - 可爱的桌面宠物陪伴您写作
- **成就系统** - 解锁各种成就获得奖励
- **等级系统** - XP积累与等级提升
- **连续写作 Streak** - 保持写作习惯

### 🎨 精美视觉设计
- **像素风 UI** - 复古像素风格设计
- **Cozy 氛围** - 温暖治愈的视觉体验
- **动态壁纸** - 10+ 精美壁纸可选
- **8种编辑器主题** - 深色/浅色/森林/海洋等主题
- **6种房间氛围** - 白天/黄昏/深夜/雨天/雪天/东京夜景

### ✍️ 强大写作功能
- **实时字数统计** - 实时显示写作进度
- **自动保存** - 永不丢失内容
- **专注模式** - 沉浸式写作体验
- **Markdown 支持** - 支持 Markdown 语法
- **大纲视图** - 方便查看文章结构

### 🏠 沉浸式房间
- **3D 立体房间** - 可交互的3D场景
- **视差效果** - 鼠标移动产生视差
- **动态灯光** - 不同时间不同光影
- **天气效果** - 雨天雨滴、雪天飘雪

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+
- npm 或 yarn

### 安装与运行

```bash
# 1. 克隆项目
git clone <repository-url>
cd pixel-writing-universe

# 2. 启动后端服务
cd backend
pip install -r requirements.txt
python app.py
# 后端服务运行在 http://localhost:5005

# 3. 启动前端服务（新开终端）
cd frontend
npm install
npm start
# 前端服务运行在 http://localhost:3000
```

### 使用说明

1. **访问首页** - 打开 http://localhost:3000
2. **进入房间** - 看到温馨的3D房间场景
3. **点击电脑** - 打开电脑屏幕
4. **开始写作** - 点击"开始码字"按钮
5. **享受写作** - 在沉浸式环境中创作

## 🎯 核心功能详解

### 1. 植物成长系统
- 7种植物可选：多肉、樱花、蘑菇、魔法植物、发光植物、藤蔓、竹子
- 5个成长阶段：种子 → 发芽 → 成长 → 开花 → 传说
- 写作字数越多，植物成长越快

### 2. 宠物陪伴系统
- 6种宠物：黑猫、柴犬、企鹅、白狐、水母、小幽灵
- 可拖拽到屏幕任意位置
- 随写作产生互动动作

### 3. 成就系统
- 10个成就等待解锁
- 奖励包括新植物、宠物、主题

### 4. 编辑器主题
- 经典深色 - 护眼深色主题
- 温暖米黄 - 温馨米黄色调
- 森林绿意 - 清新森林绿色
- 海洋深蓝 - 深邃海洋色调
- 落日橙黄 - 温暖日落色调
- 纯净白纸 - 还原真实书写
- 霓虹紫 - 赛博朋克风格
- 羊皮纸 - 复古风格

### 5. 房间氛围
- ☀️ 白天 - 明亮自然光
- 🌅 黄昏 - 橙红天空
- 🌙 深夜 - 星空月亮
- 🌧️ 雨天 - 雨滴窗户
- ❄️ 雪天 - 雪花飘落
- 🌃 东京夜景 - 霓虹灯光
